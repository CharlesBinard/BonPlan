# Custom AI Instructions — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Allow users to provide free-text instructions (global + per-search) that guide AI analysis scoring.

**Architecture:** Two new nullable text columns (users + searches), injected into analyzer user prompts as a `## User Preferences` section. Gateway schemas extended for CRUD. Frontend gets textarea + character counter in Settings, SearchCreateDialog, and SearchDetailPage.

**Tech Stack:** Drizzle ORM (migration), Zod (validation), Hono (API), React 19 + shadcn/ui (frontend)

**Spec:** `docs/superpowers/specs/2026-04-07-custom-ai-instructions-design.md`
